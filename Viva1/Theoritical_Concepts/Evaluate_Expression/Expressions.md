# Postfix and Prefix Evaluation Questions

## Postfix Expressions

1. `5 6 + 2 * 3 -`
2. `9 3 1 + * 4 -`
3. `6 2 / 3 + 4 *`
4. `7 3 2 * + 5 -`
5. `8 4 / 2 3 * +`
6. `9 2 3 * + 5 /`

---

## Prefix Expressions

1. `- * 5 6 + 3 2`
2. `+ / 9 3 - 8 4`
3. `* + 6 2 3 - 7`
4. `+ * 5 2 / 8 4`
5. `- + 9 1 * 3 2`
6. `+ / 6 2 - 5 1`

---

# Answers

## Postfix Answers

1. `(5 + 6) * 2 - 3 = 19`
2. `9 * (3 + 1) - 4 = 32`
3. `(6 / 2 + 3) * 4 = 20`
4. `7 + (3 * 2) - 5 = 8`
5. `(8 / 4) + (2 * 3) = 8`
6. `(9 + 2 * 3) / 5 = 3`

---

## Prefix Answers

1. `(5 * 6) - (3 + 2) = 25`
2. `(9 / 3) + (8 - 4) = 6`
3. `(6 + 2) * 3 - 7 = 17`
4. `(5 * 2) + (8 / 4) = 12`
5. `(9 + 1) - (3 * 2) = 4`
6. `(6 / 2) + (5 - 1) = 6`
